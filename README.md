# Vídeo IA Studio

Aplicativo web inspirado no fluxo de criação do InVideo.ai.

## O que ele faz

- Recebe um tema ou roteiro.
- Gera um roteiro curto automaticamente quando o usuário digita apenas um tema.
- Busca imagens online por palavras-chave usando URLs dinâmicas.
- Cria narração usando a voz do navegador.
- Gera legendas automáticas a partir do roteiro.
- Mostra uma prévia vertical 9:16 pronta para TikTok, Reels e Shorts.

## Como usar

Abra o arquivo `index.html` no navegador ou publique pelo GitHub Pages.

## Limitações desta primeira versão

Esta versão é um protótipo funcional em HTML, CSS e JavaScript puro. Ela não renderiza um arquivo MP4 final ainda. Para exportar vídeo real, será necessário adicionar uma etapa com backend, como FFmpeg, Remotion, Creatomate, Shotstack ou outra API de vídeo.

## Próximos passos recomendados

1. Adicionar backend para renderização MP4.
2. Integrar Pexels API ou Pixabay API.
3. Integrar ElevenLabs, OpenAI TTS ou Google Cloud TTS para narração real.
4. Salvar projetos do usuário.
5. Adicionar exportação em 9:16, 1:1 e 16:9.
